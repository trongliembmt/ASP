Đây là dự án CMS viết blog dành cho Admin và User sử dụng Angular Frontend UI có sẵn của Core UI Angular, Backend .NET 8 sử dụng swwagger để test API, Database dùng SQL SERVER.

Tab Nội Dung ( Danh mục, bài viết, loạt bài)
Khi user tạo bài viết thì sẽ lưu vào nháp. Khi user ấn mũi tên >> thì sẽ vào hàng chờ đợi admin duyệt. Admin có quyền duyệt(V) hoặc không duyệt(<<). Loạt bài có chức năng tạo các URL phù hợp với bài viết.
